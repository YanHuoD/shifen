## 一、核心流程：配料解读

```
用户打开首页
      │
      ▼
┌─────────────┐
│  看到输入框   │
│ + 3个示例按钮 │
└──────┬──────┘
       │
       ├── 点击示例按钮 ──→ 自动填入示例配料
       │                          │
       └── 手动粘贴/输入 ──────────┘
                    │
                    ▼
            ┌──────────────┐
            │ 点击「开始解读」│
            └──────┬───────┘
                   │
                   ▼
            ┌──────────────┐
            │ 展示 Loading  │
            │ (旋转图标)    │
            └──────┬───────┘
                   │
         ┌─────────┴─────────┐
         │                   │
         ▼                   ▼
   ┌──────────┐       ┌──────────┐
   │ API 成功  │       │ API 失败  │
   └────┬─────┘       └────┬─────┘
        │                  │
        ▼                  ▼
┌───────────────┐   ┌──────────────┐
│ 展示解读结果： │   │ 展示错误提示： │
│ · 健康评分     │   │ · API未配置   │
│ · 成分解读     │   │ · 网络异常    │
│ · 总体建议     │   │ · 解析失败    │
│ · 分享按钮     │   └──────────────┘
└───────┬───────┘
        │
        ▼
  ┌──────────┐
  │ 用户下一步 │
  └─────┬────┘
        │
        ├── 点击「分享到社区」
        │         │
        │         ├── 未登录 → 跳转登录页
        │         └── 已登录 → 调用 API → 显示「已分享」
        │
        ├── 修改配料文字 → 重新解读
        │
        ├── 切换到社区 Tab → 浏览帖子
        │
        └── 关闭页面 → 结束
```

## 三、社区互动流程

```
用户进入社区页
      │
      ▼
┌──────────────┐
│ 加载帖子列表  │ ← 同时恢复用户点赞状态
└──────┬───────┘
       │
   ┌───┴───┐
   ▼       ▼
有数据   无数据 → 空态引导
   │
   ▼
用户操作：
   │
   ├── 点击 💗 点赞
   │     └── 乐观更新 → API 写入 + 重算计数 → 刷新恢复状态
   │
   ├── 点击「查看成分详情」
   │     └── 展开完整成分列表 + 总体建议
   │
   ├── 点击「N 条评论」
   │     ├── 展开评论列表
   │     ├── 输入评论 → 提交（含昵称和头像）
   │     └── 删除自己的评论 → 确认弹窗 → 删除
   │
   └── 删除自己的帖子
         └── 点击垃圾桶 → 确认/取消 → API 删除（CASCADE 评论和点赞）
```

## 五、异常流程处理

| 场景 | 触发条件 | 处理方式 |
|------|----------|----------|
| AI API 未配置 | `.env` 未填 Key | 展示红色错误提示「API Key 未配置」 |
| AI 返回解析失败 | JSON 格式异常 | 展示原始返回内容的摘要 |
| AI 响应超时 | 网络慢 / API 拥堵 | Loading 持续旋转，用户可刷新重试 |
| Supabase 未配置 | `.env` 未填 URL | 登录页展示黄色提示框，社区展示示例数据 |
| 登录密码错误 | 用户输错密码 | 展示红色错误「邮箱或密码错误」 |
| 重复注册 | 邮箱已存在 | 展示「该邮箱已注册，请直接登录」 |
| 社区加载失败 | 数据库连接异常 | 展示黄色警告，降级到示例数据 |
| 分享时未登录 | 匿名用户点分享 | 跳转到登录页 |
